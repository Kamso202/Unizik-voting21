Understanding blockchain-based voting systems

Project Overview

The Blockchain-Based Voting System for University Elections is a comprehensive case study implementation designed for Nnamdi Azikiwe University. This system demonstrates how blockchain technology can revolutionize democratic processes by ensuring transparency, security, and immutability in electronic voting.

Built using vanilla HTML, CSS, and JavaScript, this system simulates a real-world blockchain voting environment while maintaining simplicity and educational value.

Case Study: Nnamdi Azikiwe University

Nnamdi Azikiwe University (UNIZIK), located in Awka, Anambra State, Nigeria, is a federal university established in 1991. Named after Nigeria's first President, Dr. Nnamdi Azikiwe, the university is committed to academic excellence and innovation.

This voting system addresses the need for transparent, secure, and efficient student union elections within the university, serving as a model for other educational institutions across Nigeria and beyond.

Blockchain Simulation

This system simulates blockchain functionality using browser localStorage, implementing key blockchain concepts including:

  • Block Structure: Each vote is stored as a block with unique hash, timestamp, and previous block reference
  • Hash Chaining: Blocks are linked through cryptographic hashes ensuring chain integrity
  • Immutability: Once created, vote blocks cannot be modified or deleted
  • Verification: Built-in tools to verify blockchain integrity and detect tampering
